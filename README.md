# 用法

### 安装 cli 工具

```
// 使用 npm 安装 CLI
$ npm install -g @tarojs/cli
# OR 使用 yarn 安装 CLI
$ yarn global add @tarojs/cli
# OR 安装了 cnpm，使用 cnpm 安装 CLI
$ cnpm install -g @tarojs/cli
```

### 下载并运行代码

```
git clone https://github.com/Wa-t/BookLendingMiniapp.git

taro start
```
